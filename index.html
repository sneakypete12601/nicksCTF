Computers only understand 1s and 0s — this is called binary. Every letter has a number (called its ASCII code), and that number can be written in binary.

For example: The letter A = 65 in decimal = 01000001 in binary.

Decode this binary message — each group of 8 bits is one character:
01000110 01001100 01000001 01000111 01111011 01100010 00110001 01110100 01110011 01011111 01100001 01101110 01100100 01011111 01100010 01111001 01110100 00110011 01110011 01111101
QUICK REFERENCE:
01000110 = F  |  01001100 = L  |  01000001 = A  |  01000111 = G
You can search online for a "binary to text converter" to help you!
The first 4 groups decode to F, L, A, G — which means the message starts with FLAG! Convert the rest of the binary groups to find the full flag. Try a free binary-to-text tool online, or work it out group by group using an ASCII table.
